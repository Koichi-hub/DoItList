# DoItList
ToDo приложение с интересной механикой. React.js + Express.js

## Backend
- Express.js
- Prisma.js
- PostgreSQL

## Перед запуском
- Установите PostgreSQL
- `cp .env.example .env`
- Измените `DATABASE_URL` в `.env`

## Запуск
- `git clone https://github.com/Koichi-hub/DoItList.git`
- `cd DoItList`
- `npm i`
- `npm run deploy && npm run start`
